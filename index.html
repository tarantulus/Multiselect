Multiselect

A jquery plugin to create searchable, sortable multi-select boxes.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to GitHub Pages.

Thanks for checking out my multiselect plugin, it creates a searchable, draggable and sortable interface for lists so you can pick multiple, non-unique values.

To use it, simply create a container (div, span, whatever you like) and then inside, two lists.


<div id="container">
		<ul id="source">
			<li>this item</li>
			<li>that item</li>
			<li>the other item</li>
			<li>the last item</li>			
		</ul>
		<ul id="dest">
			<li>empty</li>
		</ul>
	</div>

<script type="text/javascript">
	$('#source').multiselect();
</script>

The result will look something like this

  • this item
  • that item
  • the other item
  • the last item
  • empty
